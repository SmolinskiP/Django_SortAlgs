# Django_SortAlgs
Sorting algorithms on Django

Installation:<br />
Download package<br />
Install required dependencies<br />
Cd to destination folder<br />

Run ```python manage.py runserver``` in cmd/bash<br /><br />

Visit http://127.0.0.1:8000/algo/<br />
Enjoy!<br /><br />
